# VitalMed Web ⚛️

Interface administrativa do sistema VitalMed.

## 🚀 Tecnologias
- React 19 + Vite
- Material UI (MUI)
- Cypress (Testes E2E)
- Axios & JWT-Decode

## 🧪 Testes Automatizados
O projeto conta com uma suíte de testes robusta. Para rodar:
1. `npm install`
2. `npm run dev` (em um terminal)
3. `npm run test` (em outro terminal para rodar o Cypress)

## 🎨 Layout
- Dashboard com KPIs em tempo real.
- Modais intuitivos para movimentações.
- Alertas visuais para itens com estoque crítico.
